## 项目概述
- 目标：构建一个支持中英文切换的深圳旅游静态网站，具备现代化UI、视差滚动、过渡动画、图片画廊滑动与平滑导航展开，覆盖城市概览、景点、美食、交通与实用信息。
- 约束：纯 HTML5/CSS3/原生 JavaScript；响应式布局；主流浏览器兼容；页面加载优化。
- 代码注释：所有 JavaScript 函数添加中文函数级注释。

## 信息架构与页面
- 首页（`index.html`）：城市概览、特色推荐、主题色展示、语言切换入口。
- 景点介绍（`attractions.html`）：自然/人文分类网格展示，支持按类别筛选。
- 美食推荐（`food.html`）：特色餐厅与本地小吃卡片式列表，图片懒加载。
- 交通指南（`transport.html`）：地铁/公交/机场信息，常用线路示意与提示。
- 实用信息（`tips.html`）：天气/货币/紧急电话与旅行小贴士。
- 公共头尾：统一 `header`/`footer`，含导航与语言切换，跨页复用。

## 目录结构
- `assets/images/`：地标图片（平安金融中心、深圳湾大桥等）与配图。
- `assets/icons/`：SVG 图标（语言、菜单、箭头）。
- `styles/`：
  - `base.css`（重置、排版、变量与主题色：深圳蓝/绿）
  - `layout.css`（栅格与响应式断点）
  - `components.css`（卡片、导航、按钮、画廊等）
  - `animations.css`（过渡、关键帧、视差辅助类）
- `scripts/`：
  - `i18n.js`（语言存储、资源加载、DOM 文本注入）
  - `parallax.js`（滚动视差：IntersectionObserver + transform）
  - `gallery.js`（画廊滑动：CSS scroll-snap + 键盘/手势控制）
  - `nav.js`（移动端导航平滑展开/收起）
  - `main.js`（页面初始化、懒加载、性能钩子）
- `i18n/`：`zh.json` 与 `en.json`（统一键值文本资源）
- 文档：`README_DESIGN.md`（设计说明）与 `i18n-texts.md`（多语言文本对照表）

## 多语言实现
- 切换入口：导航处放置中/英切换按钮（`中文 | EN`），状态持久化至 `localStorage`。
- 文本绑定：页面元素使用 `data-i18n="key.path"` 标注；`i18n.js` 解析 JSON 并注入。
- 默认语言：读取 `localStorage`，无则按 `navigator.language` 判断（默认 `zh-CN`，否则 `en`）。
- 跨页一致：每页引入相同 `i18n.js`，保持语言选择一致。
- 对照表：`i18n-texts.md` 以表格列出所有键、中文、英文三列，便于校对。

## 视觉与主题
- 现代化风格：大留白、卡片阴影、圆角、渐变与半透明叠层（Hero 区）。
- 主色：深圳蓝（如 `#0078D4`）与城市绿（如 `#2EB872`），在 CSS 变量中统一管理。
- 地标图片：Hero 与景点卡片使用地标图；提供占位与优化指南（尺寸、压缩）。

## 动效设计
- 视差滚动：Hero 背景与前景分层，滚动基于 `IntersectionObserver` 与 `transform: translateY()`。
- 过渡动画：卡片进入/悬停使用 `transition` 与 `@keyframes`（淡入、上移、缩放）。
- 画廊滑动：横向滑动画廊，`scroll-snap` 控位，左右导航按钮与触控手势支持。
- 导航动画：移动端汉堡菜单展开/收起使用高度与透明度过渡，避免布局抖动。

## 技术实现细节
- 响应式布局：移动优先；`flex`/`grid` 实现双列/三列断点；`container` 与最大宽度约束。
- 资源组织：图片采用 WebP/JPEG 双备；`img[loading="lazy"]` 懒加载；关键图设 `aspect-ratio`。
- 性能优化：
  - JS：模块划分、`defer`/`type="module"`、避免阻塞。
  - CSS：按页面拆分与关键 CSS 内联（Hero Above-the-Fold）。
  - 图片：预加载首屏背景、压缩与响应式 `srcset`。
  - 动画：只对可见元素执行；使用 `will-change` 谨慎，避免过度合成层。
- 兼容性：Chrome/Safari/Firefox/Edge 最新版本；对 iOS 视差兼容（无需 `background-attachment: fixed`）。

## 可访问性与细节
- 语义化：`header/nav/main/section/footer`；图像 `alt`；按钮 `aria-*`。
- 键盘可用性：画廊与菜单支持键盘操作与焦点管理。
- 颜色对比与暗色适配：确保对比度合规；可选暗色模式变量。
