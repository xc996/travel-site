# 深圳旅游静态网站设计说明

- 主题色：深圳蓝 `#0078D4`、城市绿 `#2EB872`，以 CSS 变量统一管理。
- 字体与排版：Inter 作为主字体，强调现代与易读性；大标题强对比与留白。
- 组件：卡片、按钮、导航、画廊统一圆角与阴影风格；交互状态（hover/focus）明确。
- 动效：
  - 视差：Hero 背景层随滚动轻微位移，提升沉浸感。
  - 过渡：卡片进入淡入上移；菜单展开平滑。
  - 画廊：scroll-snap 结合按钮/键盘控制，移动端友好。
- 资源：建议使用 WebP/JPEG，提供 `srcset` 响应式；首屏背景可预加载。
- 可访问性：语义标签、`alt` 文本、键盘操作与焦点样式，颜色对比达标。
